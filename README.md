## 🔄 My Design Process

| 🔍 Research | 📐 Define | 🎨 Design | 🔗 Prototype | 💻 Publish |
|:-----------:|:---------:|:---------:|:------------:|:----------:|
| 사용자 이해 | 문제 정의 | UI 설계 | 인터랙션 | HTML/CSS/JS |
| 요구사항 분석 | 와이어프레임 | Figma 작업 | 사용성 검토 | 퍼블리싱 |

> 단순히 화면을 만드는 것이 아닌,
> **사용자가 자연스럽게 이해하고 오래 머무를 수 있는 흐름**을 설계합니다.
